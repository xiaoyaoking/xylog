# xylog

升级2.0版本
支持 主从日志服务器，支持单机模式， 主日志服务器仅展示， 从日志服务器为日志数据抓取。多日志节点。
新增日志自动切割。

显示nginx日志的小脚本.因为用的着急,所以写的仓促.请忽略各种语法和命名.代码是一坨屎,但是能用.
